# How It Works Section

A step-by-step walkthrough that shows new users the core workflow.

---

## Step 1: Create a Workspace

**Visual:** Screenshot of the Welcome page with the "Open Folder" button highlighted.

**Title:** Point OpenWriter at a folder

**Description:**
Select any folder on your computer as a workspace. OpenWriter will track your documents, resources, and AI context within that directory.

---

## Step 2: Write in the Editor

**Visual:** Screenshot of the rich text editor with formatted content — headings, lists, and a code block.

**Title:** Write with a powerful editor

**Description:**
Start writing in a clean, distraction-free editor. Format text with keyboard shortcuts, insert images, add code blocks with syntax highlighting, and use markdown shortcuts for speed.

---

## Step 3: Import Your Research

**Visual:** Screenshot of the Resources page with imported PDFs and documents listed.

**Title:** Add reference materials

**Description:**
Import PDFs, Word documents, and text files into your workspace. OpenWriter automatically indexes them so the AI assistant can reference them when helping you write.

---

## Step 4: Ask the AI Assistant

**Visual:** Screenshot of the AI chat panel with a prompt and a streamed response in progress.

**Title:** Get AI-powered writing help

**Description:**
Ask the AI assistant to continue your writing, research a topic, or draft new content. It searches your workspace documents and the web, then streams a grounded, reviewed response in real time.

---

## Step 5: Review and Refine

**Visual:** Screenshot showing the editor with AI-generated content integrated into the document.

**Title:** Iterate until it's right

**Description:**
The AI's built-in reflection system catches weak output before it reaches you. Review the suggestions, ask for revisions, and refine your work with AI assistance at every step.
